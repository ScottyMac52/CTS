High Contrast MFD Inserts

The MFD pages should be sized for the overlay insert slots in the Cougar MFDs.  Be sure to print out the page with 100% scaling, then trim to fit.  These high contrast JPEGs of the MFD inserts are optimized for color-blindness, but are also intended for people who fly DCS in a dark area, or just don’t want to deal with a stylized MFD insert.  These images have been scaled down to fit the browser display, so you are encouraged to use the full size images.  They can be found in the JPG folder under the folder Docs\Profile JPGs\_High Contrast MFDs, or you can simply right click on the image and select "Save Picture As." 

The modifier and TEMPO identifying conventions differ from the graphical (thematic) MFD Inserts so they can be distinguished more easily without regard to color.  The conventions are:
  • Dark Blue Bold (with Caps) is a normal button press
  • Green Italic Bold (with Caps) allows activation with TEMPO Long or S3+TEMPO Short
  • green italic lowercase requires S3+TEMPO Long for activation
  • green lowercase requires S3+button for activation
  • Blue Italic Bold (with Caps) requires TEMPO Long for activation

Note that some MFD inserts deviate from this convention (particularly the Ka-50).  Consult the notes under the MFD inserts for details.

If you have any questions, comments or suggestions, please feel free to PM “Home Fries” at CombatSim and Eagle Dynamics forums, or “HomeFries” at Combat Ace and SimHQ.

JF-17 Thunder

The 3:4 orientation of the JF-17 MFDs, along with the additional buttons for each MFD requires that TEMPO Long be utilized to command the extra buttons.  The top five vertical buttons on the main MFD display are actuated with OSB06-10 and OSB16-20, while the bottom three vertical buttons on the auxiliary MFD display are actuated with OSB08-10 Long and OSB16-18 Long.  The LMFD is the same for all users.  The JF-17 profile has different RMFD mapping based on the Number of Thrustmaster MFDs as set in CTS.  The unshifted RMFD buttons always actuate the in-game Right MFD.  For users with two MFDs (or three MFDs if the NS430 is enabled for the JF-17), holding S3 while commanding the same RMFD buttons will actuate the in-game Center MFD.

Each MFD mapping scheme requires a separate diff.lua file.  Be sure to choose the diff lua that best matches your MFD configuration.  For example, if you are running a HOTAS Cougar with two MFDs (or three MFDs with the NS430 enabled), choose TARGET JF17 HC2 v2.xx.diff.lua.  Conversely, if you have a HOTAS Warthog with three or more MFDs, select TARGET JF17 WH3 v2.xx.diff.lua.

Note that MFDs 3 and 4 are separate controllers independent of the TARGET profile.  To enable the MFD3 functionality, import the TARGET JF17 MFD3 v2xx.diff.lua to the MFD3 or MFD4 controller in the DCS Options menu.  See the section in the Setup Guide PDF  titled Controller Input Setup for details.

F-14 Tomcat

Due to the difference in nature between the Pilot and Radar Intercept Officer (RIO) stations, there are separate HOTAS and MFD controller mappings for each station.

The F-14 profile supports an additional MFD for auxiliary functions like radio and TACAN.  If you have three or more MFDs (four MFDs if you enabled the NS430 for the F-14), you can import the TARGET F14Plt MFD3 v2xx.diff.lua or TARGET F14RIO MFD3 v2xx.diff.lua to the MFD3 controller in the DCS Options menu.

F-16C Viper

The F-16C profile supports an additional MFD for use as the Integrated Control Panel (ICP).  If you have three or more MFDs (four MFDs if you enabled the NS430 for the F-16C), you can import the TARGET F16C MFD3 v2xx.diff.lua or TARGET F16C MFD4 v2xx.diff.lua to the MFD3 or MFD4 controller in the DCS Options menu ( (H/T Dr. Jester for the MFD3 diff.lua).




F/A-18C Hornet

The F/A-18C profile has two sets of MFD inserts based on the Number of Thrustmaster MFDs as set in CTS.  The LMFD in the first insert is for all users.  The RMFD in the first insert is for users with two MFDs, and the second insert includes the AMPCD and RMFD for users with at least three MFDs.  The third insert maps an MFD to the UFC, and is ideal for use with a fourth MFD.  Note that MFDs 3 and 4 are separate controllers independent of the TARGET profile.  To enable the MFD3 and 4 functionality, import the TARGET FA18C MFDx v2xx.diff.lua to the MFD3 or MFD4 controller in the DCS Options menu ( (H/T Dr. Jester for the MFD4 diff.lua).

M-2000C

The M-2000C profile supports an additional MFD to actuate VTB switches.  If you have three or more MFDs (four MFDs if you enabled the NS430 for the M-2000C), you can import the TARGET M2000C MFD3 v2xx.diff.lua to the MFD3 or MFD4 controller in the DCS Options menu.

Note: RBRT and ROSBs 11-15  in these images are context sensitive based on selected CTS options, logical Gear State, and parking brake state.  See the User Reference for an explanation of the M-2000C Direct Entry and RMFD context-sensitive buttons.

NS430 GPS


NS430 integration requires that the TARGET NS430 kb v2xx.diff.lua is installed for the NS430 module keyboard settings so that the NS430 may be brought up using the TARGET profile in all supported modules.  NS430 MFD functionality requires that a third MFD be installed on your PC.  As TARGET only programs up to two MFDs, MFD3 is an independent controller.  To enable the MFD3 functionality, import the TARGET NS430 MFD3 v2xx.diff.lua to the MFD3 controller in the DCS Options menu.  See the section in the Setup Guide PDF  titled Controller Input Setup for details.
